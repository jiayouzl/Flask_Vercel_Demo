## Python3 + Flask3 + Vercel Serverless

这是一个在 Vercel Serverless 上运行的 Python3 Flask3 应用程序的简单示例。

## DEMO

[https://jiayouzl.vercel.app](https://jiayouzl.vercel.app)

## 如何工作

此示例使用带有 Flask 的 Web 服务器网关接口（WSGI）来启用使用无服务器功能在 Vercel 上处理请求。
